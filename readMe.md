## 📦 프로젝트 사용 모듈 및 라이브러리 소개

본 프로젝트에서 사용되는 주요 파이썬 패키지들의 역할과 용도입니다.

| 모듈명 | 주요 용도 | 상세 설명 |
| :--- | :--- | :--- |
| **`requests`** | 웹 페이지 데이터 요청 | 대상 웹사이트(네이버 등)에 HTTP 요청을 보내고, HTML 소스코드나 API 응답 데이터를 서버로부터 가져오는 역할을 합니다. |
| **`beautifulsoup4`** | HTML/XML 데이터 파싱 | `requests`로 가져온 정제되지 않은 HTML 텍스트에서 원하는 리뷰 텍스트, 평점, 작성일 등의 특정 태그와 데이터를 추출(크롤링)합니다. |
| **`pandas`** | 데이터 정제 및 조작 | 크롤링한 대량의 데이터를 표(DataFrame) 형태로 구조화하고, 결측치 제거, 데이터 타입 변환, 통계 수치 계산 등 데이터 전처리를 수행합니다. |
| **`streamlit`** | 웹 대시보드 UI 구현 | 복잡한 웹 프론트엔드(HTML/CSS/JS) 지식 없이도, 파이썬 코드 몇 줄만으로 크롤링 데이터 시각화 결과를 멋진 웹 애플리케이션으로 만들어 배포할 수 있게 합니다. |
| **`plotly`** | 대화형(Interactive) 시각화 | 평점 분포 차트, 날짜별 리뷰 추이 그래프 등을 생성합니다. 마우스를 올리면 상세 수치가 표시되는 다이내믹하고 직관적인 시각화 툴입니다. |

### 🛠️ 한번에 설치하기
터미널(Terminal) 환경에서 아래 명령어를 실행하여 위 모듈들을 한 번에 설치할 수 있습니다.

```bash
pip install requests beautifulsoup4 pandas streamlit plotly
```

## 📦 구글 연동 관련 추가 모듈 및 라이브러리 소개

Google Sheets API 연동 및 보안 인증을 위해 새롭게 추가된 파이썬 패키지들의 역할과 용도입니다.

| 모듈명 | 주요 용도 | 상세 설명 |
| :--- | :--- | :--- |
| **`gspread`** | 구글 시트 데이터 제어 | 파이썬 코드로 구글 스프레드시트를 직접 제어할 수 있게 해주는 라이브러리입니다. 시트 생성, 데이터 읽기/쓰기(`update`), 내용 비우기(`clear`) 등의 작업을 수행합니다. |
| **`oauth2client`** | 구글 API 보안 인증 및 권한 부여 | 구글 클라우드 플랫폼(GCP)에서 발급받은 서비스 계정 키(JSON)를 기반으로, 데이터에 안전하게 접근할 수 있도록 OAuth2 인증 및 보안 토큰을 생성하고 관리합니다. |

### 🛠️ 추가 모듈 설치하기
터미널(Terminal) 환경에서 아래 명령어를 실행하여 구글 연동에 필요한 라이브러리를 설치할 수 있습니다.

```bash
pip install gspread oauth2client
```
